# AI足球赛事研判指挥舱

静态 GitHub Pages 客户端，包含：

- 500彩票网竞彩足球日常自动更新
- Interwetten五年历史联赛画像与500近期增量
- 多Agent圆桌、实时阵容新闻、爆冷防线、反防平、主推与冷门比分脚本
- 一眼看懂统一结论与历史研判回看
- 客户手动录入主队、客队和比赛时间
- 浏览器本地授权激活与演示模式

## 自动更新

GitHub Actions 在北京时间 08:15、14:15、18:15 运行 `tools/update_daily_data.py`。抓取失败时保留上一版赛事数据，不会写入空列表。

## 使用范围

仅提供体育数据分析、赛前研究与内容创作参考；不涉及赌博，不提供下注服务，不承诺结果。
